# 🛒 Zepto SQL Data Analysis Project

## 📌 Overview
This project analyzes Zepto's product inventory dataset using pgAdmin 4. The objective is to clean the data, perform exploratory data analysis (EDA), and answer real-world business questions using SQL. The project demonstrates SQL skills such as filtering, aggregation, grouping, sorting, CASE statements, and data transformation.

## 📂 Dataset
The dataset contains product information including:
- Product Name
- Category
- MRP
- Discount Percentage
- Discounted Selling Price
- Available Quantity
- Weight
- Stock Availability

## 🛠️ Tools Used
- pgAdmin 4
- SQL

## 📊 Project Workflow
1. Created the `zepto` table with appropriate data types.
2. Explored the dataset by checking row count, null values, duplicate products, and unique categories.
3. Cleaned the data by removing invalid records and converting prices from paise to rupees.
4. Performed business analysis using SQL queries.

## 🔍 Business Questions Solved
- Top 10 products with the highest discounts
- High-MRP products that are out of stock
- Estimated revenue by category
- Products with high MRP and low discounts
- Categories with the highest average discounts
- Price per gram analysis for better value comparison
- Product weight classification (Low, Medium, Bulk)
- Total inventory weight by category

## 📈 SQL Concepts Demonstrated
- DDL & DML
- SELECT, WHERE, ORDER BY
- GROUP BY & HAVING
- Aggregate Functions
- CASE Statements
- DISTINCT
- Data Cleaning (UPDATE & DELETE)

## 🚀 Learning Outcomes
This project strengthened my understanding of SQL for data analysis, data cleaning, aggregation, and extracting actionable business insights from real-world retail data.
